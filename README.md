# Hangman Game

## Objective

This is a simple Hangman Game developed using Python.

The computer randomly selects a word, and the player guesses one letter at a time. The player has six chances to guess the correct word.

## Features

- Random word selection
- Letter-by-letter guessing
- Input validation
- Remaining attempts counter
- Win/Lose message

## Technologies Used

- Python 3

## How to Run

1. Install Python.
2. Download the project.
3. Open terminal in project folder.
4. Run:

```bash
python hangman.py
```

## Concepts Used

- Loops
- Lists
- Strings
- Random Module
- Conditional Statements
